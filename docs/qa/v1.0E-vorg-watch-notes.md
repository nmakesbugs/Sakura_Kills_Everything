# v1.0E — Vorg Watch (QA Note)

Short, by design.

## 1. Episode 4 adds a new evidence-board dynamic
Four episodes, four toys: Bird = timed strike, Squirrel = suspect monitoring, Rabbit = chase
map, **Vorg = evidence-board investigation**. The win for this build: *"this one is the weird
mystery episode."*

## 2. The Vorg remains unconfirmed
Every theory's final **Vorg Status** is an "Unconfirmed" variant (asserted by a dedicated test
that also forbids a bare "confirmed"). The joke is that concern rises while confirmation never
does.

## 3. The toy is classification + concern, not solving
The player examines **3 of 5** clues (Bent Grass, Fence Shadow, Unexplained Rustle, Suspicious
Leaf, Unknown Track), classifies each (Vorg Evidence / Squirrel Deception / Rabbit Misdirection
/ Wind-Leaf Activity / Inconclusive But Troubling), and the comic **Concern Level** meter climbs
(Calm → Concerned → Officially Concerned → Unhelpfully Alarmed → Vorg Watch Activated) no matter
what — because Sakura. It is deliberately **not a deduction puzzle**; all paths lead to
questionable official concern. Then the player files one of four theories.

## 4. Outcome families (4 theories)
- **Probable Vorg Activity → *Watch Activated*** — likely reality: "The grass was bent and
  everyone became dramatic."
- **Squirrel Cover-Up → *Cover-Up Suspected*** — "A squirrel was nearby, which is not technically
  evidence."
- **Wind Conspiracy → *Atmospheric Concern*** — "A leaf moved."
- **Unknown, Therefore Serious → *Case Remains Open*** — "No one knows what happened, including
  Sakura."

Each report carries official/likely, a Tanisha note, the reviewed-evidence list, the concern
level (in the case line), and the Vorg Status. Screenshot-worthy.

## 5. Tanisha should react to the far-corner absurdity and the report
She is the **reality anchor**: setup witness beat, the mid-investigation complication ("Sakura,
it is probably a bag."), and a final report note she declines to fully endorse. The board is the
toy; Tanisha keeping a straight face next to it is the joke.

## 6. Future episodes should keep changing the toy
Four distinct toys now exist. Episode 5 should invent its own (a stealth meter, a tug-of-war, a
stakeout timer), not reuse one. The shell + report make that cheap.

## Efficiency
Reused the shared shell + report card verbatim; the evidence board (markup/CSS/logic) is **local**
to `episodes/vorg-watch/`. No mystery engine, no storage/archive. Shell, legacy platform,
`src/*`, and the other episodes untouched. Index/root each got one card; one new spec added.

## How to play
Open `episodes/vorg-watch/index.html` (or via the index / root). ~3–5 min first play. Mobile
portrait (412×760). Seam: `window.__tc.reachReport('vorg'|'squirrel'|'wind'|'unknown')`.

## Versioning
HUD label **v1.0E**; `package.json` → `1.0.0`. This is **v1.0 of the episodic prototype
direction** (four playable episodes, each a different toy) — explicitly **not** a final
commercial release.
