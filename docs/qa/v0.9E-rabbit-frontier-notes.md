# v0.9E — Rabbit at the Garden Frontier (QA Note)

Short, by design.

## 1. Episode 3 adds a different playable dynamic
Episodes 1–2 use scene → choice → timed action → report. Episode 3 keeps the comic structure
but swaps the **middle toy** for a small **chase map**. The win condition for this build was
*"the player should feel: this one plays differently."*

## 2. The middle toy is a chase map
A 5-node backyard graph — Patio, Garden, Grass, Bushes, Fence Line. Sakura starts at the patio,
the rabbit near the garden. Over **3 rounds** the player picks Sakura's route (glowing reachable
nodes; concrete copy like "Cut through the grass" / "Charge the bushes"); the rabbit then flees
on a weighted graph that biases toward the fence. After round 2 there's a small **complication**
(Tanisha calls from the window; a leaf moves — investigate it or stay on mission). A final
**lunge** (commit / hesitate / overcommit) closes it out.

## 3. It is intentionally simple
No hidden rules, no stat screens, no real tactics. Distance and round are shown as chips. Even
"failure" produces a good report. The toy exists to give Episode 3 a different *physical rhythm*,
not to be a strategy game.

## 4. Outcome families (4)
- **Strategic Intercept** → *Garden Secured*
- **High-Speed Overcommit** → *Force Applied*
- **Rabbit Escape** → *Suspect Fled*
- **Leaf Diversion** → *Investigation Expanded*

Each yields a distinct official interpretation / likely reality / Tanisha note / evidence /
**Rabbit Status** (e.g. *Escaped, under protest*; *Temporarily beyond jurisdiction*; *Unobserved
(see: leaf)*) and a route summary in the case line. **No confirmed rabbit catch** — the rabbit
always escapes/retreats/eludes, consistent with the others.

## 5. Tanisha should react to the chase and the report, not to systems
She gets the setup witness beat, a **mid-chase line** ("Sakura…" from the window), and a final
report note. The map is the toy; Tanisha is still the point.

## 6. Canon / lore
Light seasoning only — Garden Frontier, fence line, "border violation", jurisdiction. No
lore-dump; you don't need any canon doc to play.

## 7. If it still feels samey
Then future episodes need **stronger toy differentiation** — the shell + report are now reusable
enough that the per-episode interaction is where freshness has to come from. Push the toy
further (a stealth meter, a tug-of-war, a stakeout timer), not the chrome.

## Efficiency
Reused the shared shell and report card verbatim; map markup/CSS/logic are **local** to
`episodes/rabbit-frontier/`. Did not touch the shell, the legacy platform, `src/*`, or the other
episodes. Index/root got one new card each; one new spec added.

## How to play
Open `episodes/rabbit-frontier/index.html` (or via the index / root). ~3–5 min first play.
Mobile portrait (412×760). Test seam: `window.__tc.reachReport('intercept'|'overcommit'|'escape'|'leaf')`.
